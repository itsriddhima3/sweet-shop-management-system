Sweet Shop Management System
A full-stack application for managing a sweet shop inventory, built with a modern tech stack following Test-Driven Development (TDD) principles.

Features
User Authentication: Register and login with JWT-based authentication

Sweet Management: CRUD operations for managing sweets (Admin only)

Inventory Control: Purchase and restock sweets with real-time quantity updates

Search & Filter: Find sweets by name, category, or price range

Responsive Frontend: Modern, mobile-friendly interface

Tech Stack
Backend
Node.js with Express (JavaScript)

MongoDB database with Mongoose ODM

JWT for authentication

Jest for testing

Frontend
React (JavaScript)

Tailwind CSS for styling

React Router for navigation

Axios for API calls
